# 📈 EarningWhisperer (어닝 위스퍼러)

개인 투자자를 위한 실시간 어닝콜 AI 스트리밍 분석 및 모의 매매 시스템입니다. 
기관 투자자의 전유물이었던 실시간 NLP 스코어링 기술을 개인의 브라우저 위로 가져와 정보 비대칭성을 해소합니다.
