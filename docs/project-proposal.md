# 📈 EarningWhisperer: AI 기반 어닝콜 실시간 분석 및 모의투자 자동매매 시스템

## 1. 프로젝트 개요 (Overview)
**EarningWhisperer**는 기업의 실적 발표(Earning Call) 텍스트 스트림을 실시간으로 분석하여, 알고리즘 매매를 위한 시그널을 생성하고 모의투자 환경에서 자동 매매를 수행하는 웹 서비스입니다. 연간 수천만 원에 달하는 기관 투자자들의 전유물이었던 '실시간 정보 기반 매매'를 일반 개인 투자자의 브라우저 위로 가져와 정보 비대칭성을 해소하는 것을 목표로 합니다.

## 2. 기획 배경 및 시장 분석 (Background & Market Analysis)

### 📊 시장 상황 팩트체크
* **B2C (개인용 서비스의 한계):** 토스증권이 자체 AI를 통해 미국 주식 어닝콜 실시간 번역을 제공하고 있으며, 여러 주식 유튜버들이 라이브 동시통역을 진행하고 있습니다. 하지만 이는 **'단순 번역 및 인간의 주관적 해설'**에 머물러 있습니다.
* **B2B (기관용 AI 플랫폼의 독점):** 블룸버그 터미널(연간 약 3천만 원)이나 AlphaSense 같은 B2B 플랫폼은 단순 번역을 넘어 실시간 NLP 감성 분석, 키워드 트래킹, 그리고 이를 알고리즘 매매 엔진에 직접 연결하는 기능을 금융 기관들에게 제공하며 압도적인 속도 우위를 점하고 있습니다.
* **경쟁사 분석 (Gap Analysis):** 현재 글로벌(Composer, Trade Station) 및 국내(젠포트, 퀀트킹) 핀테크 플랫폼들이 AI를 적극 도입하고 있으나, 이는 모두 장 마감 후의 **'사후 분석'**이나 **'중장기 매매 전략 생성'**에 집중되어 있습니다. **개인 투자자를 위한 '실시간 어닝콜 AI 스트리밍 분석 및 자동매매 시스템'은 전무한 상황**입니다.

### 💡 EarningWhisperer의 핵심 가치 (Why Real-time AI?)
이러한 시장 상황 속에서 본 프로젝트는 다음 세 가지 핵심 니즈를 충족합니다.
1. **"정보의 비대칭성" 해소 (속도의 우위):** 기관 투자자들만 누리던 실시간 NLP 스코어링 기술을 도입하여, 개인 투자자도 발표 즉시 시장의 방향성을 읽고 대응할 수 있게 합니다.
2. **"행간의 의미(뉘앙스)" 파악 (비정형 데이터 분석):** 단순 번역으로는 알 수 없는 경영진의 어조, 긍정/부정의 강도를 AI가 정량적인 데이터(-1.0 ~ +1.0)로 변환하여 제공합니다.
3. **초고변동성 이벤트에서의 "감정 배제":** 어닝콜 직후의 급등락장에서 패닉 셀링이나 뇌동매매를 방지하고, 철저히 데이터와 AI 시그널에 기반한 규율 있는 매매(모의투자)를 실행합니다.

## 3. 타겟 유저 및 포지셔닝 (Target Audience & Positioning)

**포지셔닝 전략:** *"가장 빠르고 객관적인 기관급 <실시간 어닝콜 AI 분석기>를 개인 투자자의 안방으로"*

본 서비스는 단순히 뉴스를 소비하는 것을 넘어, 주도적으로 시장에 참여하는 다음 두 그룹을 핵심 타겟으로 삼습니다.

* **확신이 필요한 '재량적 트레이더 (Discretionary Trader)':** 어닝콜 발표 순간의 변동성을 노리고 직접 매매를 진행하지만, 자신의 판단이 맞는지 빠르고 객관적인 근거(AI 시그널)를 통해 심리적 확신을 얻고자 하는 적극적인 투자자.
* **팩트 체크가 필요한 '스마트 개미 (Smart Retail Investor)':** 주식 유튜버나 인플루언서의 편향된(Biased) 해석에 휘둘리지 않고, 어닝콜 원문 텍스트와 객관적인 AI 감성 분석 데이터를 직접 교차 검증하여 투자에 반영하고 싶은 분석형 투자자.

## 4. 핵심 기능 (Core Features)
1. **실시간 어닝콜 데이터 파이프라인:**
   * 슬라이딩 윈도우 방식을 적용하여 문맥 단절 없이 실시간 텍스트 데이터를 수집 및 처리.
2. **하이브리드 AI 매매 엔진 (Two-Track):**
   * **[속도] 스코어링 엔진:** FinBERT를 Teacher-Student 방식으로 파인튜닝하여, 텍스트 입력 즉시 수 밀리초 내에 긍정/부정 매수 강도(Raw Score)를 추출.
   * **[설명력] 해설 엔진:** 스코어링 엔진이 도출한 점수(방향성)를 바탕으로, LLM이 유저 친화적인 해설(Rationale)을 비동기적으로 생성하여 논리적 환각을 차단.
3. **백엔드 매매 로직 및 리스크 관리:**
   * 지수이동평균(EMA)을 활용한 시계열 점수 누적 및 노이즈 필터링 로직 (Java).
   * 모의투자 환경에서의 주문 실행 및 포트폴리오 비중 조절(방어 로직).
4. **실시간 대시보드 (Web UI):**
   * 실시간 어닝콜 스크립트, 감성 점수 추세선, 최종 매매 시그널 및 LLM 해설을 직관적인 UI로 제공.

## 5. 기술 스택 (Tech Stack)
### 🧠 AI & Data Pipeline
* **Language:** Python 3.10+
* **Framework:** FastAPI
* **NLP / LLM:** HuggingFace `transformers` (FinBERT), OpenAI API 
* **Broker:** Redis (Pub/Sub)

### ⚙️ Backend
* **Language/Framework:** Java 17+, Spring Boot
* **Database:** MySQL (User, Trade History, Portfolio), Redis (Session, Real-time Data)

### 💻 Frontend
* **Framework:** React.js / TypeScript
* **State & Network:** Zustand, Socket.io (WebSocket)
